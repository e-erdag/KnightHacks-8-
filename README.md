# Code Cards - KnightHacks-8- Hackathon
Clash Royale Inspired Programming Tutor.

 ## Setup Instructions

1. Install Node.js

2. Clone the Repository
```bash
git clone https://github.com/your-username/KnightHacks-8-.git
cd KnightHacks-8-
```
3. Install Dependencies
```bash
npm install
```
4. Run the Program
```bash
npm run dev
```
